Cómo Obtener el ID de Envío

  1. Inicia sesión en tu revista OJS
  2. Ve a "Envíos" → "Mis Envíos"
  3. Busca el envío que quieres usar
  4. En la lista verás un número como 2415 (ese es el ID)
  5. También puedes verlo en la URL: ...submissionId=2415

API del Bot

Token para API: {{ admin_config.bot_token }}

Endpoints Disponibles:

GET /api/journals
curl -H "X-Bot-Token: {{ admin_config.bot_token }}" {{ request.host_url }}api/journals
Lista todas las revistas configuradas
POST /api/journals
curl -X POST -H "Content-Type: application/json" -H "X-Bot-Token: {{ admin_config.bot_token }}" -d '{"name":"Mi Revista","host":"https://...","username":"user","password":"pass","default_submission_id":"123"}' {{ request.host_url }}api/journals
Agrega una nueva revista
POST /api/upload
curl -X POST -H "Content-Type: application/json" -H "X-Bot-Token: {{ admin_config.bot_token }}" -d '{"journal_id":"abc123","submission_id":"2415","links":["https://.../file1.pdf","https://.../file2.docx"]}' {{ request.host_url }}api/upload
Inicia una subida de archivos

  • /update_journal id campo valor
  • /delete_journal id
  • /upload journal_id submission_id link1 link2...
  • /status task_id - Estado de subida
  • /report journal_id - Genera reporte
